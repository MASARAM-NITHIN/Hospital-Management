# 🏥 Hospital Management System (Spring Boot)

A full-stack **Hospital Management System** developed using **Spring Boot**, designed to manage patients, doctors, and appointments efficiently. This project demonstrates a clean architecture with proper separation of concerns and real-world application design.

---

## 🚀 Features

* 👤 Patient Management (Add, View, Delete)
* 👨‍⚕️ Doctor Management
* 📅 Appointment Scheduling
* 🔗 Entity Relationships (Patient ↔ Doctor ↔ Appointment)
* 🗄️ MySQL Database Integration
* 🌐 Thymeleaf-based dynamic frontend
* 🎨 Responsive and interactive UI with CSS

---

## 🛠️ Tech Stack

* **Backend:** Spring Boot, Spring MVC, Spring Data JPA
* **Frontend:** HTML, CSS, Thymeleaf
* **Database:** MySQL
* **Build Tool:** Maven
* **Language:** Java

---

## 📂 Project Structure

* `controller` → Handles HTTP requests
* `service` → Business logic layer
* `repository` → Database operations
* `entity` → JPA entities (tables)
* `templates` → HTML pages (UI)

---

## 📊 Key Highlights

* Follows **MVC Architecture**
* Uses **Hibernate ORM** for database mapping
* Implements **CRUD operations**
* Designed as a **resume-level project**
